# Photography Website

Статичен фотографски сайт, направен с HTML, CSS и JavaScript.

## Файлове
- `index.html` — структурата на сайта
- `style.css` — цялата визия и responsive дизайн
- `script.js` — меню, sticky header и демо submit съобщение
- `assets/` — изображенията, които изпрати

## Стартиране
Отвори `index.html` директно в браузър или качи цялата папка в хостинг.
